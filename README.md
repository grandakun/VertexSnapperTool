# 顶点吸附工具（Vertex Snapper）

简介
----
`顶点吸附工具.py` 是一个运行在 Autodesk Maya 中的交互式工具，用于将选定顶点吸附到参考顶点或参考模型的顶点位置。工具提供“自动吸附”和“手动吸附”两种模式，并带有简洁的 UI，适合在建模修缝、清理贴合或对齐顶点时快速使用。

主要功能
----
- 自动吸附：将当前选择的顶点吸附到已加载的参考模型的对应或最近顶点位置；参考模型通过“加载选中物体为源模型”按钮设置。
- 手动吸附：允许用户手动加载一组参考顶点（作为目标位置）和一组要移动的顶点，然后执行吸附操作。
- 基于距离的最近点匹配逻辑：工具为每个目标顶点找到距离最近的参考位置并移动过去。
- UI 操作简单，支持撤销，并在命令行打印处理统计信息。

主要接口与函数
----
- `VertexSnapperUI`（类）
  - 工具的 UI 与状态封装，包含以下方法：
    - `show()`：全局函数，创建并显示工具窗口。
    - `load_auto_source_model()`：在自动模式下加载参考模型（一个 transform，必须包含 mesh）。
    - `execute_auto_snap()`：在自动模式下执行吸附（会把选中顶点移动到参考模型顶点的世界位置）。
    - `load_manual_reference_verts()` / `clear_manual_reference_verts()`：手动模式下加载/清除参考顶点列表。
    - `load_manual_verts_to_move()` / `clear_manual_verts_to_move()`：手动模式下加载/清除要移动的顶点列表。
    - `execute_manual_snap()`：在手动模式下执行吸附操作。
    - `_perform_snap_logic(verts_to_move, reference_positions)`：工具的核心逻辑，实现按最近参考位置将顶点移动到目标位置，包含 undo 分块处理。

UI 说明
----
- 窗口名称：`vertexSnapperUIWindow`
- 标签页：
  - 自动吸附（Auto Snap）
    - 步骤1: 加载选中物体为源模型（会读取源模型的所有顶点位置作为参考位置）
    - 步骤2: 选择要移动的顶点并点击“吸附选中顶点到源模型”执行
  - 手动吸附（Manual Snap）
    - 步骤1: 加载参考顶点（提供目标位置）并可清除
    - 步骤2: 加载要移动的顶点并可清除，点击“执行吸附”开始移动

使用示例（在 Maya Script Editor）
----
1) 直接启动工具（推荐放入 Shelf）

```python
import sys
sys.path.append(r"y:\Code\Library")
import 顶点吸附工具
reload(顶点吸附工具)  # 如果在 Maya 中已加载过脚本，可使用 reload
顶点吸附工具.show()
```

2) 自动吸附典型流程

 - 在场景中选择参考模型（单个 transform）并点击“加载选中物体为源模型”。
 - 选择需要被移动的顶点（组件选择，例如 `pSphere1.vtx[0:10]`）。
 - 点击“吸附选中顶点到源模型”。

3) 手动吸附典型流程

 - 选择一组参考顶点（可以来自任意网格），点击“加载源顶点”。
 - 选择一组目标顶点（要移动的顶点），点击“加载目标顶点”。
 - 点击“执行吸附”。

行为与限制
----
- 对每个目标顶点，工具使用简单的欧氏距离计算找到最近的参考位置并移动目标顶点到该位置；不做拓扑或映射匹配，适用于需要快速对齐的场景。
- 工具会在 `_perform_snap_logic` 中打开 undo 分块，所有移动操作可通过 Maya 的 undo 恢复。
- 如果参考顶点或目标顶点被删除或不存在，工具会给出警告并跳过。
- 对大量顶点操作时，移动性能受顶点数量和 Maya 响应速度影响，建议分批次处理大型网格。

扩展建议
----
- 增加匹配策略选项：例如按顶点索引顺序匹配、按最近面的投影、按法线方向过滤等。
- 支持权重插值：对移动的顶点应用平滑权重或按距离插值，避免突变。
- 增加“最大匹配距离”阈值，超出该距离的顶点不被移动并列入报告。
- 提供批量脚本接口，使工具可在无 UI 的流水线中以脚本方式调用。
